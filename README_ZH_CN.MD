# ChatGPT 问题导航侧边栏

为 ChatGPT 官网提供了一个便捷的问题侧边栏目录。它能够自动搜集当前会话页面的用户提的问题，并展示在侧边栏上，提供快速导航到历史问题的位置的能力。
